🎯 Recommendation Widget Service

✅ Сервис работает!

Доступные endpoints:

  • POST /api/identify - Идентификация пользователя
  • POST /api/track - Трекинг событий
  • GET /api/recommendations/:clientId - Получение рекомендаций
  • GET /api/admin/sites - Управление сайтами
  • GET /admin - Админ-панель

Установка виджета:

<script>
  window.widgetConfig = {
    apiUrl: 'https://' + window.location.host,
    siteDomain: window.location.hostname,
    themeColor: '#3B82F6',
    title: '🎁 Рекомендуем для вас'
  };
</script>
<script src="https://' + window.location.host + '/widget.js" defer></script>

🎉 Ваш сервис успешно развернут!
