🎯 Recommendation Widget Service

Виджет рекомендаций для вашего сайта на основе поведения пользователей

✅ Сервис работает

🚀 Быстрый старт

Добавьте этот код на ваш сайт:

<!-- Добавить перед закрывающим тегом body -->
<script>
  window.widgetConfig = {
    apiUrl: 'https://menu-vidget.vercel.app',
    siteDomain: window.location.hostname,
    themeColor: '#3B82F6',
    title: '🎁 Рекомендуем для вас',
    position: 'bottom-right'
  };
</script>
<script src="https://menu-vidget.vercel.app/widget.js" defer></script>

📡 API Endpoints

POST /api/identify - Идентификация пользователя
POST /api/track - Трекинг событий
GET /api/recommendations/:clientId - Получение рекомендаций
